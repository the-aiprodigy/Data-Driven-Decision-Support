# V2 Executive Delivery

Files:
- `upfront_pricing_precision_v2.ipynb`: notebook version using pandas/numpy/matplotlib/seaborn.
- `run_analysis_v2.py`: dependency-light production script (standard library only) that runs in restricted environments.
- `executive_report_v2.md`: concise business report with top opportunities and recommendations.
- `analysis_summary_v2.txt`: computed KPI summary generated by `run_analysis_v2.py`.
- `switch_rate_by_*_v2.csv`: segment-level outputs generated by `run_analysis_v2.py`.

## How to run
From repository root:

```bash
python 'Taxi Upfront Pricing Precision/v2_executive_delivery/run_analysis_v2.py'
```

This command produces executable outputs in the same folder for stakeholder review and downstream automation.
